# Frontend Testing
 
 This directory is reserved for frontend tests.
 
 ## Test Strategy
 
 -   **Unit Tests**: Use Jest and React Testing Library for component testing.
 -   **Integration Tests**: Test page interactions and API integration flow.
 -   **E2E Tests**: Use Playwright or Cypress for end-to-end user flows (e.g., creating a campaign).
 
 ## Running Tests
 
 (Once tests are implemented)
 
 ```bash
 npm test
 ```
